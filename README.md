# 大老二 Big Two撲克牌遊戲

# Intro your game
大老二是由四個玩家進行的撲克牌遊戲，每個玩家在遊戲開始會拿到13張牌，拿到梅花三的玩家可以優先出牌，其他玩家則依順時針出牌比牌，但一定要出比上一家還大的牌組，如沒有比上家大的牌組，則失去該輪出牌權力，如果玩家讓其他三家都無法出牌，則下一輪可以出任意牌型，最先把手中所有牌組出完的玩家優勝。
